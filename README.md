# News Mag

A responsive news aggregator web application displaying real-time news across categories
like politics, technology, sports, health, and entertainment.

## Features
- Real-time headlines via News API
- Category-based browsing (politics, tech, sports, health, entertainment)
- Search and filter functionality
- Fully responsive UI

## Tech Stack
React.js · Bootstrap · News API

## Setup
```bash
git clone https://github.com/hariom0205/News.git
cd News
npm install
npm start
```

## Environment Variables
Add your News API key to a `.env` file:
```
REACT_APP_NEWS_API_KEY=your_api_key_here
```
